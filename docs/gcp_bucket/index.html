PETIT-GAN

This is a repository containing all the artifacts involved in the research of Omri Berman's M.Sc., conducted in the optical-sensing lab of Iftach Klapp.
For more details about the research, please visit the project's website.

The repository artifacts are as follows:

  • data: The dataset involved in the research.
  • demo: artifacts used by the demo flow in the PETIT-GAN Github repository.
  • docs: research-related documentation.
